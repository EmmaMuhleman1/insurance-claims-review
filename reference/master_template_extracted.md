## FORMATTING PATTERN SUMMARY

### Sub-Header Labels (within report body)
These appear as **bold + underline** in Normal style, directly below the List Paragraph bullet header:
- `Cost Analysis`
- `RCV`
- `ACV`

### Vendor Proposal Lines
- Format: **BOLD**, ALL CAPS
- Pattern: `[VENDOR NAME] ([PHONE]) PROPOSAL DATED ON [MM/DD/YYYY]`

### Key Phrases Always Used
| Phrase | Context |
|--------|---------|
| "reportedly" | Always used before damage descriptions (legal protection) |
| "fair and reasonable" | Used when affirming costs/amounts |
| "pre-loss condition" | Used in RCV statement |
| "linear age depreciation of the materials only" | Used in ACV statement |
| "We submit that" | Standard assertion opener |
| "Loss Solutions Group submits" | Formal assertion opener |
| "the claimed event" | Standard reference to the loss event |
| "A further breakdown of the pricing is available in the attached spreadsheet" | RCV section |
| "A further breakdown of this is available in the attached spreadsheet" | ACV (varies) section |

### Number Formatting Convention
- Ages and life expectancies are written as: word form (digit form)
- Examples: "two (2) years old", "twenty (20) years"
- Dollar amounts: always with two decimal places and comma separators: "$58,650.00"

### Report Structure Order
1. Salutation: `Dear [Title + Name],`
2. Opening: Fixed thank-you paragraph
3. - Assignment: scope description
4. - Findings and Analysis: insured contact, vendor contact, total presented
5. - Scope of damage: equipment + loss summary
6. - Age and Condition: equipment age (variants: nameplate, insured-reported, new/unused)
7. - Cause of loss: (variants: weather, vandalism/theft, other)
8. - Reparability: repair vs. replacement justification
9. - Cost Analysis: methodology + vendor proposal review + components + labor
10. - RCV: replacement cost value statement
11. - ACV: actual cash value statement (variants: single life, varies, new/unused skip)
12. - Salvage: (variants: Hartford referral vs. standard "little to no salvage")

### Conditional Logic Map

| Condition | Affected Section | Variant |
|-----------|-----------------|---------|
| Equipment age known from nameplate | Age and Condition | Variant 1: "provided with photographs...nameplate" |
| Equipment age reported by insured | Age and Condition | Variant 2: "The insured reported..." |
| Equipment is new/unused | Age and Condition | Variant 3: "new and unused" |
| Equipment is new/unused | ACV | SKIP: "has not completed an ACV analysis...new and unused" |
| Equipment is new/unused | Spreadsheet | Remove ACV from spreadsheet |
| Weather/wind loss | Cause of Loss | Variant 1: NOAA + Weather Underground confirmation |
| Vandalism/theft loss | Cause of Loss | Variant 2: "photos...information reported...consistent with" |
| Single equipment type | ACV | Variant 1: state specific life expectancy |
| Multiple equipment types | ACV | Variant 2: "varies based on equipment type" + spreadsheet |
| Carrier = Hartford | Salvage | "refer to your internal salvage department" |
| Carrier != Hartford | Salvage | "little to no salvage value" |
| Repair path | Reparability | "proposed repair path is warranted" |
| Replacement path | Reparability | "proposed replacement path is warranted" |
